# 3D Graphics - Camera and Lighting

Built using C++ and OpenGL

## Features
1. Renders 10 cubes
2. Each cube has a wood texture
3. Move around with W-A-S-D and mouse using the **custom built Fly Camera**
4. Flashlight and four point-light sources
